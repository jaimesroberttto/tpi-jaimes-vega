1. PRESENTACION INICIAL

Hola, somos Roberto Jaimes y Gabriel Vega.
En este video vamos a presentar nuestro Trabajo Practico Integrador de Programacion 1,
titulado "Gestion de Datos de Paises en Python: filtros, ordenamientos y estadisticas".

El objetivo de este proyecto fue desarrollar una aplicacion en Python que trabaje con datos
de paises a partir de un archivo CSV, permitiendo realizar altas, actualizaciones, busquedas,
filtros, ordenamientos y calculo de estadisticas.


2. EXPLICACION GENERAL DEL PROBLEMA

La consigna nos pedia crear un sistema en consola que permita gestionar informacion sobre
paises. Cada pais contiene los siguientes datos:

- nombre
- poblacion
- superficie
- continente

La informacion se guarda en un archivo CSV y se carga al iniciar el programa.

Nuestro sistema fue pensado para aplicar los conceptos vistos en la materia:

- listas
- diccionarios
- funciones
- condicionales
- estructuras repetitivas
- manejo de archivos CSV
- filtros y ordenamientos
- estadisticas


3. ESTRUCTURA DEL PROYECTO

Los archivos principales del proyecto son:

- main_paises.py
- funciones_paises.py
- paises.csv
- README
- informe teorico

En `main_paises.py` se encuentra el menu principal y la interaccion con el usuario.
En `funciones_paises.py` esta toda la logica del sistema:

- carga del CSV
- guardado del CSV
- validaciones
- filtros
- ordenamientos
- estadisticas

El archivo `paises.csv` contiene el dataset base con paises de ejemplo.


4. EXPLICACION DE LA ESTRUCTURA DE DATOS

Para representar la informacion utilizamos una lista de diccionarios.

Cada pais se almacena como un diccionario con claves:

- nombre
- poblacion
- superficie
- continente

La lista permite recorrer todos los paises facilmente, mientras que el diccionario permite
acceder a cada dato de forma clara por nombre de campo.

7. EXPLICACION TECNICA BREVE

En este proyecto utilizamos:

- listas de diccionarios para los datos en memoria
- funciones para modularizar el programa
- csv.DictReader para leer el archivo
- csv.DictWriter para guardar el archivo
- sorted() para realizar ordenamientos
- condicionales y bucles para controlar el flujo del menu

Tambien incorporamos validaciones de entradas y control de lineas invalidas en el archivo CSV.
